## Course Project — Multimodal Tool Retrieval

We propose a multimodal retrieval system for agentic tool-use, treating natural language (user queries) and structured data (API schemas, code snippets) as distinct modalities. Using the ToolBench dataset, we explore three ideas: **Hierarchical Contrastive Alignment**, a **Code-Enhanced Tri-Encoder**, and **Hard-Negative Mining** from agent decision-tree traces.
