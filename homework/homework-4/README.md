# Homework 4 — GRPO: Reinforcement Learning for Vision-Language Models

**Course:** Multimodal AI (MAS.S60 / 6.S985) · Spring 2026 · MIT

## Overview

This assignment covers reinforcement learning for Vision-Language Models, implementing and training a VLM using Group Relative Policy Optimization (GRPO).

## Topics Covered

- GRPO vs. PPO: group-based advantage estimation without a critic
- Reward design and reward hacking
- Implementing GRPO advantage computation from scratch
- Training a VLM with GRPO using TRL
- Post-training evaluation on held-out images
